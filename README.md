# YAHIB Cabinet Tools

## Overview

YAHIB Cabinet Tools is a professional SketchUp extension for designing and manufacturing melamine furniture.

The extension is developed to automate cabinet production by generating drilling, connectors, and manufacturing data.

## Version

Current Version: 0.1.0 (Planning Stage)

## Main Goals

- Minifix 15 Automation
- Confirmat Support
- Dowel Support
- Cabinet Generator
- Cut List Generator
- Edge Banding List
- Hardware List
- Assembly Drawings
- CNC Export
